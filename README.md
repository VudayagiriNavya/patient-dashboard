# patient-dashboard
Patient Dashboard Web App

## 📌 Description
A web-based dashboard to manage patient data and visualize health records.

## 🚀 Features

* View and manage patient records
* Display patient details in a structured dashboard
* Search and filter patient information
* Interactive user interface for easy navigation
* Responsive design for better usability


## 🛠️ Technologies Used

* HTML5 – for structuring the patient dashboard interface
* CSS3 – for styling and layout design
* JavaScript – for adding interactivity and dynamic behavior

## ▶️ How to Run
1. Download or clone the repository to your local system.
2. Navigate to the project folder.
3. Open the `index.html` file in any web browser (such as Google Chrome).
4. The Patient Dashboard application will load and be ready to use.
